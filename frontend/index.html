About This Project

Thesis Project - Computer Science & Engineering

Project Overview

This Bangla Braille to Voice conversion system is developed as a final year thesis project to demonstrate the application of deep learning in accessibility technology. The system aims to bridge the communication gap for visually impaired individuals by converting Braille text to audible speech.

Technical Architecture

CNN/Transformer Model

Advanced neural network for Braille character recognition

Text-to-Speech

Google TTS engine optimized for Bengali language

Research Methodology

Machine learning research with extensive testing
